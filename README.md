# Student Management System (Java)

## Description
This is a simple Java-based menu-driven application to manage student records using classes, constructors, instance members, and an ArrayList.

## Features
- Add Student
- Display Students
- Search by PRN
- Search by Name
- Update Student
- Delete Student
